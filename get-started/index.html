Installation on your server

Setting up Seatsurfing on your Linux server just takes a few minutes.

  1. Start the Docker container:
    docker run \
    --rm \
    -p 8080:8080 \
    -e "POSTGRES_URL=postgres://seatsurfing:DB_PASSWORD@db/seatsurfing?sslmode=disable" \
    seatsurfing/backend
  2. Open the administrative web interface at and log in with: admin@seatsurfing.local / 12345678
    http://localhost:8080/admin/
  3. Invite your coworkers to the Booking Web App which is accessible via mobile devices and computers at:
    http://localhost:8080/ui/

For more information, please check out the documentation.
